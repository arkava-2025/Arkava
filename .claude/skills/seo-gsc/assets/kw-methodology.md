# Metodología: Keywords de Oportunidad desde Google Search Console

## Definición

Keywords de oportunidad son consultas donde el sitio ya tiene visibilidad en Google
(impresiones + posicionamiento) pero aún no alcanza su máximo potencial de tráfico,
CTR o conversión. Google ya reconoce la URL — solo falta optimización.

## Características para clasificar una keyword como oportunidad

1. Posición entre 5 y 20
2. Mínimo 5 impresiones en el período analizado
3. CTR por debajo del benchmark de su posición
4. Intención de búsqueda alineada al negocio
5. URL posicionando no es la más relevante para esa query (canibalización)

## Tipos de oportunidad

### Tipo A — Cercanas al Top 5 (posición 5-7)
- Acción: optimizar title tag y meta description con diferenciador comercial
- Impacto esperado: +2-4 posiciones en 4-6 semanas
- Incluir: precio, ciudad, año, propuesta de valor única

### Tipo B — Segunda página (posición 8-15)
- Acción: profundizar contenido + agregar FAQ + mejorar estructura H2
- Impacto esperado: entrar al Top 10 en 6-10 semanas

### Tipo C — CTR bajo para su posición
- Acción: reescribir snippet para aumentar click-through
- Señal: el ranking está bien pero el título/descripción no genera clicks
- Fix: añadir números, preguntas, palabras de acción, localización

### Tipo D — URL incorrecta posicionando
- Acción: crear landing específica + 301 redirect o actualizar contenido de la URL actual
- Señal: un blog informativo posicionando para una query transaccional
- Fix: crear página de producto/servicio dedicada y redirigir autoridad

## Proceso de extracción mensual

1. Exportar datos GSC: últimos 90 días, dimensiones query + page
2. Filtrar: posición 5-20, mínimo 5 impresiones
3. Calcular CTR gap (esperado - real)
4. Clasificar por tipo (A/B/C/D)
5. Priorizar: posición × impresiones × intención comercial
6. Asignar acción concreta a cada keyword
7. Ejecutar y medir en 4 semanas

## Framework de priorización

| Factor | Peso |
|---|---|
| Posición (más cerca de 5 = mayor prioridad) | 30% |
| Impresiones | 25% |
| Intención comercial (transaccional > informacional) | 20% |
| CTR gap (cuánto se está perdiendo) | 15% |
| Historial de conversión de esa URL | 10% |

## Errores comunes a evitar

- Atacar solo keywords nuevas ignorando las que ya tienen visibilidad
- No revisar si el CTR mejoró después de optimizar el snippet
- Crear contenido nuevo sin revisar si ya hay una URL que posiciona
- Ignorar la canibalización (dos URLs compitiendo por la misma query)
- No alinear la intención de búsqueda con el tipo de página que se crea
- Medir menos de 4 semanas después de un cambio (Google tarda en recalibrar)
